Визуализация вылета #

Обработчик "HotA.CanHaveBattles" берёт id объекта (0..4), умножает на размер записи 19 байт и прибавляет начало вектора из поля подсистема+0x10. Объект подсистемы цел, id корректен, но вектор записей пуст - его начало (begin) равно нулю, поэтому чтение поля записи [NULL + id*19 + 0x0D] падает по нулевому указателю. Пустой вектор - штатное состояние, не порча.

A: объект подсистемыEAX, валиден
поле +0x10begin вектора
B: begin = NULLвектор пуст

начало вектора (подсистема+0x10) прочитано успешно и равно 0 (вектор пуст)

B = 0база NULL
id*19id из {0..4}
F: чтение [+0x0D]адрес = id*19+0x0D

адрес сбоя = 0 + id*19 + 0x0D (0x0D, 0x20, 0x33, 0x46, 0x59 по логам)

A Объект подсистемы выделен (EAX - правдоподобный адрес кучи); его поле +0x10 должно хранить базовый адрес таблицы записей.
B Начало вектора (begin) равно NULL - вектор записей пуст (штатное состояние, не порча); загрузчик ещё не добавил записи.
F Адрес сбоя = база + id*19 + 0x0D. Так как база = 0, адрес чтения совпадает ровно с id*19 + 0x0D - чистое разыменование нуля.
